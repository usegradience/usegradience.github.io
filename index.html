What's Gradience?

Customize LibAdwaita with ease

  • Gradience only modify CSS
  • Gradience can be extended with plugins for customizing more things like Firefox GNOME Theme
  • Each Gradience theme (called preset) can be exported and shared as JSON
  • Gradience Team provide an official repo with verified themes
